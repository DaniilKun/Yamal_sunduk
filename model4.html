СУМКА ДЛЯ ХРАНЕНИЯ ПРЕДМЕТОВ
РУКОДЕЛИЯ

Сумка в виде мешка овальной формы, со шнурком, затягивающим сумку. Сумка сшита из черной кожи, верхняя часть из темно-розовой замши. Сумка украшена металлическими пуговицами, подвесками, цепочками. К верхней части сумки пришит чехол для наперстка в виде круглого кармана, украшенного круговым узором из разноцветного бисера и коричневого меха.

ПЕРИОД СОЗДАНИЯ: ХХ век

МАТЕРИАЛ, ТЕХНИКА: кожа, оленья замша, мех оленя, бисер, шитье вручную

РАЗМЕР: 39x33 см
