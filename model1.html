«ПАД»
МЕШОК ДЛЯ ХРАНЕНИЯ ВЕЩЕЙ

Мешок для хранения вещей - мешок овальной формы сшит вручную из меха оленя, верхняя и боковая части вшиты из оленьей замши. В верхней части мешок заужен. С одной стороны, мешок украшен узором из чередующихся квадратов темного и белого меха оленят с копытец, орнаментом «голов- ки», прошивками из красного, синего, оранжевого, зеленого сукна. Задняя часть мешка украшена треугольными вставками из белого меха. С обеих сторон мешка имеются завязки.

ПЕРИОД СОЗДАНИЯ: ХХ век

МАТЕРИАЛ, ТЕХНИКА: сукно, мех оленя, оленья замша, шитье вручную.

РАЗМЕР: 50x44 см.
