Кукла Нухуко

Кукла Нухуко (нен.)- Кукла изготовлена из широкого утиного клюва с треугольником, нарисованным белой краской, кукла сшита из сукна зеленого, желтого, красного цвета. К сукну пришит фрагмент орнамента из желтого сукна. По верхнему краю пришита светло-зеленая тесьма.

ПЕРИОД СОЗДАНИЯ: конец ХХ века

МАТЕРИАЛ, ТЕХНИКА: клюв, сукно, бисер, шитье вручную

РАЗМЕР: 11,3x8,5 см.

ВЫБРАТЬ ДРУГУЮ ВЕЩЬ
