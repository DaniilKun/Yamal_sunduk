ТУЦЯ (НЕН.) - СУМКА ДЛЯ ХРАНЕНИЯ
ШВЕЙНЫХ ПРИНАДЛЕЖНОСТЕЙ

Мешок круглой формы. Сшит из оленьего меха и замши. Мешок, с одной стороны, украшен прошивками из красного сукна, вставками из узорных полос. По нижнему краю украшен бахромой из замши. Верхняя часть из оленьей замши, заужена. В верхней части пришиты две пары замшевых шнуров.

ПЕРИОД СОЗДАНИЯ: конец ХХ века

МАТЕРИАЛ, ТЕХНИКА: мех оленя, сукно, шитье вручную

РАЗМЕР: 15x13 см
